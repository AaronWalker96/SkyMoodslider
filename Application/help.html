User Guide

When you open the application for the first time, you'll be able to see the Moodslider page with all the sliders in their default positions and the content suggestion sections will show “No content”.

Uploading Content

To add programmes to the application, navigate to the “Upload Content” page and select the "Choose a file" button. You can then select a file from your local machine that contains a list of programmes, the file must be in a JSON format. Once you have selected your file, you can then click the "Upload" button to upload your document to the application and you'll be returned to the Moodslider page.

Using the Moodslider

To use the Moodslider, move the tab of each slider to match your current feelings. The application will then present an array of programmes tailored to your selected mood profile. You can even move multiple sliders at the same time to be presented with programmes that match multiple moods.
